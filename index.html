Day 1 上午：演說驅動商業模式

一、未來 500 強思維：你不一定要上市，但要有上市的思維

核心觀念：

  • 公司要能「自動運轉」──不依賴單一個人。
  • 上市、出海不是必須，但要有資格選擇。
  • 擁有「可預測性、可複製性、可投放性」的系統，才能持續成長。

啟發案例：米其林指南

  • 本質不是做美食，而是讓人開車更遠 → 輪胎使用率增加。
  • 倒果為因：透過改變使用情境，讓目標受眾自然增加需求。
  • 啟示：思考如何拆解顧客行為，創造更多使用場景。

二、商業模式與品牌定位

聚焦但要有定位：

  • 沒有定位，就只是雜亂的生意。
  • 領頭羊的差異：不是產品最好，而是「讓人第一個想到你」。
  • 甜點案例：建立品牌目的＝「大家買甜點就會想到 Kiwimu」。

最好的商業模式：

  • 可系統化、可擴大。
  • 能被不同人複製，而不是依靠個人天賦。

三個關鍵角色（每個企業必備）：

  1. 做產品／服務／內容的人。
  2. 領域專家（能解決問題、建立信任）。
  3. 把東西賣出去的人（懂市場與商業模式）。

三、三大核心技能：流量 → 銷售 → 產品／服務

1. 流量（Traffic）

  • 觸及 → 名單 → 聯絡方式
  • 原則：一定要留下名單，就算當下沒成交，也能未來轉化。
  • 工具：廣告投放、A/B 測試、漏斗設計。
  • 指標：
    • CPC（點擊成本）
    • 名單獲取率（CF 30%、EB 50%為標準）

2. 銷售（Sales）

  • 關鍵在於「轉換率」。
  • 演講、簡報、訪談：讓客戶快速理解價值。
  • 消費者心理：專業≠成交，必須找到他們的「行動動機」。

3. 產品／服務（Product/Service）

  • 不只是專業，而是「能否被理解、被選擇」。
  • 價值階梯設計：
    • 演說（最高價值影響力）
    • 衍生性產品（延伸體驗）
    • 洞察性產品（透過洞察設計需求）
    • 複製型產品（標準化、擴張用）

四、數據與自動化的力量

  • 客戶獲取成本（CAC）與 名單獲取成本（CPL）是商業判斷基礎。
  • 透過自動化銷售漏斗（行銷工具、系統留言、自動跟進），確保效率。
  • 用數據預測未來，而不是憑感覺。

五、公眾演說的價值

  • 學會演說 = 掌握媒體影響力。
  • 分享 > 販售：透過故事、洞察、教育，讓人「自願加入」。
  • 招不到人，不是因為缺人才，而是你的事業沒有吸引力。

六、核心心法

  • 財富不是給予的，而是創造出來的。 → 自由也是一樣，需要主動設計。
  • 心法 / 策略 vs 技術：技術能解決眼前問題，但心法與策略才是可持續成長的根本。
  • 定位優先，專業其次：先讓人認識你、記住你，再來強調專業。

七、應用與延伸（針對你的品牌 / Kiwimu）

品牌定位

讓消費者一想到「甜點的情感陪伴」，就先想到 Kiwimu。

價值階梯規劃

  • 頂層：情緒 × 故事的演說課程（甜點結合自我覺察）。
  • 中層：聯名商品、角色體驗（Kiwimu 情緒禮盒）。
  • 底層：標準化的招牌甜點（生日蛋糕、禮物蛋糕）。

流量策略

  • 短影音 + IG/FB 廣告測試 A/B。
  • 每次活動一定要收集名單（LINE、Email）。

演說策略

  • 將 Kiwimu 的故事變成「品牌演說」：
    • 甜點不是商品，而是「陪伴的情緒體驗」。
    • 讓人從演說 → 共鳴 → 記住 → 消費。

👉 總結

Day1 上午的核心，不是單純學演講，而是學 如何用演說驅動商業模式。

用演說製造流量與信任 → 引導到銷售 → 最後承接到產品／服務。

一切以 名單、數據、定位 為基礎。

分享的力量 > 技術的力量。

🎤 演講設計藍圖（以終為始）

一、結尾設計（CTA）

  • 原則：功能 > 功用 > 情緒價值
  • 在結尾前，先列出 3 個讓對方得到的好處，語言要能帶動情緒，讓聽眾感覺「非你不可」。

👉 範例模版：

「我會提供 功能X → 幫助你 功用X → 讓你 滿足情緒X」

「我會提供 功能Y → 幫助你 功用Y → 讓你 滿足情緒Y」

「我會提供 功能Z → 幫助你 功用Z → 讓你 滿足情緒Z」

範例：

  • 室內裝修：丈量 → 家人都有舒服的空間；設計製圖 → 先滿足每個人對家的想像；裝潢施工 → 用100萬打造出300萬的價值
  • 音樂教育：師資調度 → 幫你省下30萬師資成本；多元課程 → 招生率提升50%；發表安排 → 曝光度增加，家長信任
  • 企業端合作：代工服務 → 節省3–4千萬成本；包裝設計 → 省掉買包裝機的投資；批量製造 → 沒有庫存壓力

二、演講三大框架

  • 簡介（收斂）：
    • 開場：一句話交代你是誰、做什麼
    • 不要一開始就「專業術語」，用人話先把結果講出來
  • 主體內容（聚焦）：
    • 把專業轉換成聽眾要的「結果」
    • 功能→功用→情緒價值
    • 透過故事、案例讓聽眾自己投射進來
  • 成交環節（延展）：
    • 重申「三個好處」
    • CTA：讓聽眾知道「下一步該怎麼做」

三、故事設計（IP × 人性投射）

架構：WWHHE

  • WHEN：從什麼時間點開始
  • WHAT：發生了什麼事（痛點、困難）
  • WHY Q：為什麼必須解決
  • HOW：你是怎麼解決的
  • END：最後得到什麼結果

👉 套用範例（以甜點IP為例）：

WHEN：當我第一次要設計月島甜點的時候

WHAT：發現市場上甜點同質性太高，客人只看價格

WHY Q：如果不找到差異化，我的品牌就會被淹沒

HOW：於是我決定用「角色故事 + 情緒投射」來打造IP

END：結果不只客人買甜點，更是因為故事與角色留下來

四、語言技巧

  • 專有名詞要減少：避免距離感
  • 形容詞動詞化：不是「專業的設計」，而是「設計讓你看見未來的家」
  • 語氣與聲調：重點詞放慢，收尾提高，讓情緒帶動成交

✅ 總結：一場演講的關鍵

  • 先想結尾 → 三大好處
  • 再設計主體 → 把專業翻成人話，結果導向
  • 最後鋪陳故事 → 讓客戶投射「我也能得到相同結果」

📖 公眾演說千萬成交影響術 Day 2

一、思維核心

  • 知識的重量 × 行動的能量：學得越多，越能感覺自己的不足。真正有價值的不是知識本身，而是能否落地行動。
  • 定位與價值：「在其位，謀其政」：要先給自己一個位置（定位），市場才會給你價值。商業是市場的驗證。產品 / 服務只有被市場認可，才真正有意義。

二、專利與 IP

申請專利

  • 國際專利一百萬美金（高昂成本）。
  • 原則：確定 IP 能產生價值 → 再去申請專利。
  • 例：新加坡國際專利授權案例。

應用啟發

  • 對於品牌 / IP，要先透過市場驗證價值（銷售、授權、合作）→ 再用專利 / 版權保護。
  • 避免「空燒錢」，確保投入有商業意義。

三、巨星行銷法 STAR(S)

營造百萬品牌的秘訣

  • S – Social Media (社群媒體)：建立社群影響力，持續曝光。
  • T – Traditional Media (傳統媒體)：
    • 進入報章雜誌、電視等大眾媒體。
    • 傳統媒體帶來流量與「權威感」。
    • 用「謙虛地自誇」方式：讓別人替你曝光，而不是自己誇。
  • A – Author (作者身份)：
    • 出書是最強名片，擁有稀有的公信力與收藏價值。
    • 書籍是「權威象徵」＋「永久傳播工具」。
    • 方法：自己寫 / 編輯 / 設計 / 出版 / 推廣；集中火力在某一天製造「暢銷榜第一」效應；電子書可快速更新內容 → 高效率曝光。
    • 實戰話術：「你好，我是《XXX》這本書的作者，這本書最近成為 Amazon 暢銷榜第一名……」
  • R – 與巨星同行：透過合作、同台、合影 → 轉嫁巨星光環。
  • S – 成為講師（價值10萬美金的超級加速器）：
    • 站上舞台即是「身份地位」的象徵。
    • 透過講座可收名單、建立權威，而不是直接銷售。

四、關鍵字策略

1. 永久性關鍵字（長期價值）

  • 國家、語言、藝術娛樂作品（電影、音樂、書籍）
  • 「如何」
  • 名人或明星名字

2. 季節性關鍵字（提前佈局）

春節、聖誕節、情人節、生日、奧運、世界杯、考試、畢業季、選舉……

  • 策略：事件發生之前就先布局，等事件爆發時收割流量。

3. 煙花式關鍵字（短期爆發）

新冠病毒、AI演算法、元宇宙等

  • 爆發快 → 消失快。

4. 應用原則

  • 先做季節性 → 再做永久性 → 偶爾利用煙花式。
  • 精準比流量更重要。

五、媒體感知與說服

媒體感知：訊息投放後，觀眾的感受才是重點。

如何建立信任：

  • 客觀且真實的角色形象。
  • 國際曝光（媒體、論壇）。
  • 與激勵人心的價值掛鉤。

高流量文案公式 P.R.E.P

  • Punch/Power – 有力度的訊息
  • Relatable – 讓人感同身受
  • Easy to understand – 易懂
  • Probability – 提供可信概率 / 數據

六、價值階梯（Value Ladder）

財富方程式 = 一對多影響 × 品牌

  • 引流產品（$0）：免費試用、社群體驗。
  • 低價產品（無利產品）：製造差異性，建立粉絲基礎。
  • 中低價產品（微利產品）：向上銷售 / 向下加購。
  • 中高價產品（讓利產品）：提供更多價值與差異化。
  • 高價產品（結果導向）：資源、人脈、專屬結果。適用於願意付出高成本的少數人。
  • 最終極服務 = 培訓 / 教學：讓客戶學會使用產品，並且願意持續付費。

七、如何為高價值服務定價

  • 參考產業標準。
  • 結果價值（幫助客戶省時間、省錢、省力，提升形象、權威、舒適、收入）。

七大價值槓桿：

  • 提升形象
  • 提升權威
  • 提升舒適感
  • 提升收入
  • 幫客戶省錢
  • 幫客戶省時間
  • 幫客戶省力

八、故事行銷術（Lion）

1. 畢卡索案例

  • 自我宣傳：雇人製造「市場好奇心」。
  • 名人代言：用簽名支票取代現金。
  • 專業社交：用專業（畫肖像）交換人脈。
  • 創造競爭：資訊不對稱，製造抬價空間。
  • 異業結盟：跨界合作（芭蕾舞劇）。

2. 盈利故事架構

  • 目標 → 阻礙 → 行動 → 結果
  • 強化版：目標 → 阻礙 → 努力 → 結果 → 意外轉彎 → 結局

3. 故事應該包含

  • 情感色彩與感官細節
  • 三種層次：別人的故事（社會驗證）；自己的故事（真實案例）；團隊的故事（共同使命）
  • 三種時間：過去、現在、未來
  • 三種面向：失敗、成功、感動

九、文化行銷

  • 對內行銷：讓團隊與內部人員理解品牌價值。
  • 對外行銷：一致對外，傳達同樣的文化與理念。
